HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страницы. CSS (Cascading Style Sheets) — это язык стилей, который определяет внешний вид элементов HTML. Вместе они позволяют создавать красивые и функциональные веб-страницы.

Ненумерованный список

  • HTML - язык разметки гипертекста
  • CSS - каскадные таблицы стилей
  • JavaScript - язык программирования для веб-страниц
  • Flexbox - модель размещения элементов
  • Grid - система двумерного макета
