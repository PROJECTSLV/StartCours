Что такое переменные?

Переменные - это контейнеры для хранения данных. Представьте их как коробки с названиями, в которых можно хранить разные значения.

// Объявление переменных
let имя = "Анна";
let возраст = 25;
let активен = true;

Ключевые слова: let, const, var

let - для переменных, которые можно изменять
const - для констант (неизменяемых значений)
var - устаревший способ (лучше не использовать)

// let - можно изменять
let счетчик = 0;
счетчик = 1; // Это работает!

// const - нельзя изменять
const PI = 3.14;
// PI = 3.15; // Ошибка!

Типы данных

JavaScript автоматически определяет тип данных. Основные типы:

// Строки (текст)
let имя = "Мария";
let сообщение = 'Привет!';

// Числа
let возраст = 20;
let цена = 99.99;

// Логические (true/false)
let студент = true;
let спит = false;

// undefined - неопределенное значение
let неизвестно;

// null - пустое значение
let ничего = null;

Полезные советы

  • Используйте осмысленные имена переменных (например, userName вместо x)
  • Для констант используйте UPPER_CASE (например, MAX_USERS)
  • Всегда объявляйте переменные перед использованием
  • Используйте console.log() для вывода информации и отладки
  • Не бойтесь экспериментировать и делать ошибки!
